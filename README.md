# Codesapce

A repository for exploring and testing GitHub Codespaces and Copilot features.

## Documentation

- [GitHub Copilot Modes Guide](GITHUB_COPILOT_MODES.md) - Comprehensive guide explaining Local, Background, and Cloud modes in GitHub Copilot

## Contents

This repository contains various example files for testing Codespaces functionality:
- GPU examples (TensorFlow and general)
- Machine learning examples (Random Forest)
- Algorithm implementations (Heapsort in C)
- Jupyter notebooks for limit testing
